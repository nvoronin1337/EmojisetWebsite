Current Projects


Emojiset Mining

The Emojiset Mining team has created a website that can analyze emojisets from tweets and infer the meaning behind them. The code is capable of pulling tweets given various settings, and can either provide a small or large data set, which can be sent to users via email. A user can test the website using a demo, but must acquire a Twitter developer account and sign up to access all of the features.


Epidemiology

The team has implemented a new model named Biased Majority rule model which consists in model that resembles how opinions spread through majorities. The model contains 3 states: Adopter, Rejector and Undecided and includes a bias parameter that can be modified by the user.
